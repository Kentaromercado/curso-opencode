# Curso Opencode

Este es un proyecto de ejemplo para aprender a usar Opencode.

## Descripción

Opencode es una herramienta de IA que permite interactuar con diferentes proveedores de modelos de lenguaje a través de una interfaz de terminal (TUI).

## Contenido del curso

- Instalación de Opencode
- Configuración de proveedores de IA
- Uso de la interfaz TUI
- Gestión de sesiones
- Comandos avanzados

## Requisitos

- Git
- Node.js
- npm

## Instalación

```bash
npm install -g opencode-ai
```

## Uso

```bash
opencode --help
```
